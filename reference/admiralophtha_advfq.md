# Visual Function Questionnaire Analysis Dataset

An example Visual Function Questionnaire (VFQ) analysis dataset

## Usage

``` r
admiralophtha_advfq
```

## Format

An object of class `tbl_df` (inherits from `tbl`, `data.frame`) with 972
rows and 93 columns.

## Source

Derived from the ADSL and QS datasets using `{admiral}`,
`{admiralophtha}` and the [ADVFQ
template](https://github.com/pharmaverse/admiralophtha/blob/main/inst/templates/ad_advfq.R).
The full, open-source VFQ questionnaire can be accessed
[here](https://www.nei.nih.gov/about/education-and-outreach/outreach-materials/visual-function-questionnaire-25).
